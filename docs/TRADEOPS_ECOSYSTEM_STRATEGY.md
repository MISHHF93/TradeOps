# TradeOps Ecosystem Strategy

## Positioning

**The AI Commerce Operating System for modern merchants** — the intelligent orchestration layer **above** the global commerce ecosystem.

TradeOps does **not** compete with Shopify, BigCommerce, WooCommerce, Amazon, eBay, Alibaba, Stripe (merchant payments), Google Merchant Center, Meta, TikTok, ShipStation, EasyPost, carriers, PayPal, QuickBooks, or Xero.

It **increases their value** by making merchants more successful, catalogs higher quality, sourcing more qualified, and operations more reliable.

## Guiding question

> How do I run my commerce business better?

## Ecosystem flywheel

```text
Merchant success
  → more GMV on channels
  → more qualified supplier orders
  → better feeds for ads / Merchant Center
  → more fulfillment demand
  → more SaaS subscription value for TradeOps
  → more AI automation depth
  → better merchant success
```

## Partner success metrics (honest)

Track only measurable, org-scoped (or platform-scoped) facts:

* merchant product/case throughput  
* connector utilization and health  
* listing and order completion rates  
* payment normalization coverage  
* AI operator run outcomes  
* never fabricate partner GMV for unconnected platforms  

See `GET /api/v1/ecosystem/partners` and `/terminal/ecosystem`.

## Build principle

Every feature must answer:

1. Which lifecycle stage?  
2. Which Commerce Case?  
3. Which business capability?  
4. Which ecosystem participants benefit?  
5. What is the next action if empty/blocked?
