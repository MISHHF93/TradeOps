# Partner Value Framework

## Rule

Every TradeOps capability should create **measurable value** for at least one ecosystem participant **without** replacing that participant’s platform.

| Partner | Value | TradeOps mechanism |
|---------|-------|--------------------|
| Merchant | Better decisions, less manual work | Intelligence + process + AI |
| Shopify / storefronts | Higher quality merchants & GMV readiness | Catalog prep, ops discipline |
| Amazon / eBay | Better listings & operational reliability | Canonical listing + policy gates |
| Suppliers / Alibaba | Qualified buyers | Evaluate → source with margin gates |
| Stripe | SaaS subscription volume | Platform billing |
| PayPal / marketplaces | Settlement volume (merchant-owned) | Payment intelligence only |
| Google Merchant | Cleaner feeds | Artifact + content readiness |
| Meta / TikTok | Campaign-ready catalogs | Media + content optimization |
| ShipStation / carriers | Optimized shipping demand | Fulfillment stage + connectors |
| TradeOps | SaaS + enterprise + AI automation revenue | Entitlements |

## Partner Success Center

* API: `GET /api/v1/ecosystem/partners`  
* UI: `/terminal/ecosystem`  
* Honesty: registry-only partners never show fake GMV  

## Anti-patterns

* Building a rival storefront  
* Forcing merchants off Shopify Payments  
* Pooled capital products as core GTM  
* Silent simulated “live” partner stats
